# 🎯 ProjectMind - Complete User Guide

## 📖 What is ProjectMind?

**ProjectMind** is your AI-powered productivity copilot that helps you complete long-term projects on time. Think of it as having a smart assistant that:

- 📋 Plans your project tasks automatically
- 👀 Watches your coding activity
- 🤖 Gives you AI-powered insights
- ⚡ Tracks your productivity in real-time
- 🎯 Keeps you focused and on track
- 🔗 Connects with your GitHub commits

**Perfect for:** Students working on semester projects, developers building side projects, freelancers managing client work, or anyone who struggles to finish what they start.

---

## 🚀 Getting Started (5 Minutes)

### **Step 1: Create Your Account**

1. Visit the ProjectMind website
2. Click **"Sign Up"**
3. Enter your email and password
4. Click **"Create Account"**

✅ You're in! Now let's create your first project.

### **Step 2: Create Your First Project**

1. Click **"+ New Project"** button
2. Fill in the details:
   - **Project Name:** "My Portfolio Website"
   - **Goal:** "Build a personal portfolio with React and deploy it"
   - **Deadline:** Select your target date
   - **Hours per Day:** How many hours you can work daily (e.g., 3 hours)

3. **(Optional)** Upload documents:
   - Upload your project requirements (PDF, DOCX, TXT)
   - Upload design mockups or specifications
   - AI will read these to understand your project better

4. Click **"Create Project"**

🎉 **Magic happens!** AI analyzes your goal and generates a complete task breakdown with time estimates.

### **Step 3: Start Working**

1. Open your project dashboard
2. See your tasks organized by day
3. Start coding!
4. ProjectMind automatically tracks your activity

## ❓ Frequently Asked Questions

### **Q: How does ProjectMind track my activity?**
**A:** Your IDE/editor sends activity events (edits, saves, opens, commits) to ProjectMind. It's automatic - no manual logging needed. If you connect GitHub, commits are also tracked.

### **Q: Is my code sent to ProjectMind?**
**A:** No! Only metadata is sent (file names, event types, timestamps). Your actual code stays on your machine. GitHub commits are analyzed by AI, but code content is not stored.

### **Q: How accurate is the AI?**
**A:** AI provides confidence scores (0-100%) with every decision. High confidence (80%+) is very reliable. Lower confidence means AI is less certain - use your judgment.

### **Q: Can I edit AI-generated tasks?**
**A:** Yes! AI generates initial tasks, but you can:
- Edit task titles
- Change time estimates
- Adjust priorities
- Add/remove tasks
- Reorder tasks

### **Q: What if I disagree with AI recommendations?**
**A:** That's fine! AI provides suggestions, not commands. You're in control. Use AI insights as guidance, but trust your own judgment.

### **Q: How is productivity calculated?**
**A:** Four scores averaged:
- **Daily (30%):** Active time today
- **Focus (30%):** Deep work sessions
- **Consistency (25%):** Days active
- **Intensity (15%):** Coding velocity

### **Q: Why is my risk level high?**
**A:** Check "AI Reasoning & Explanations" to see exactly why. Common reasons:
- Behind schedule
- High-priority tasks incomplete
- Low productivity
- Approaching deadline

### **Q: Can I use ProjectMind without GitHub?**
**A:** Yes! GitHub is optional. You can still:
- Create projects
- Get AI task planning
- Track activity (if IDE integration available)
- Use all AI features
- Get productivity insights

### **Q: Does ProjectMind work offline?**
**A:** No, ProjectMind requires internet connection for:
- AI features (Groq API)
- Real-time updates (Socket.IO)
- Data sync (MongoDB)

### **Q: Can I have multiple projects?**
**A:** Yes! Create unlimited projects. Each project has its own:
- Task list
- Activity tracking
- Productivity metrics
- AI insights
- GitHub integration

### **Q: How do I delete a project?**
**A:** From the projects list, click the delete icon (🗑️) on any project card. Confirm deletion. This cannot be undone.

### **Q: Can I share my project with others?**
**A:** Currently, projects are private to your account. Team collaboration features are planned for future releases.
