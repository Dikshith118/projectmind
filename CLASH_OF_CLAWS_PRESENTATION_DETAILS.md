**Model:** Groq - llama-3.3-70b-versatile
**Why Groq?** FREE, fast (14,400 requests/day), powerful

**AI Features:**

1. **Project Planning**
   - Input: Project name + description/document
   - AI generates: Tasks, estimates, dependencies
   - Prompt: "Break down this project into tasks with time estimates"

2. **AI Copilot**
   - Input: User question + project context
   - AI provides: Intelligent, context-aware answers
   - Prompt: "Based on current project state, answer: {user_question}"

3. **Commit Analysis**
   - Input: Git commit message + diff
   - AI categorizes: Feature/Bug/Refactor/Docs
   - Prompt: "Analyze this commit and categorize it"

4. **Delay Detection**
   - Input: Task activity patterns
   - AI predicts: Risk level, delay probability
   - Prompt: "Analyze task activity and predict delays"

5. **Recovery Planning**
   - Input: Current project state + delays
   - AI generates: Recovery strategies
   - Prompt: "Generate recovery plan for delayed project"

6. **Productivity Insights**
   - Input: Activity logs, commit history
   - AI provides: Insights, recommendations
   - Prompt: "Analyze productivity patterns and provide insights"

**Framework:** Custom AI reasoning engine with:
- Context builder (gathers project data)
- Prompt templates
- Response parser
- Fallback handling
